# Ring and Bully Algorithm

## Steps to run Ring program

- Open new terminal

```bash
cd 6_ring_bully
```

- Compile the file

```bash
javac *.java
```

- Run the file using

```bash
java Ring
```

```bash
⚠️ Make sure java jdk version is 1.8.x using java -version
```
